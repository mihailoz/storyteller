Game Rules


Storywriters is a typing-based game made for multiple players.
Players take alternating turns, typing only one word at a time.
Each player has 10 seconds to type the word.
If the player decides to end the sentence, he should put a punctuation mark, but he can't write anything else that turn.
The goal of the game is to write a story and have fun doing so.
The game ends when the majority of players decide the story should be finished (by clicking the end-game vote button).
