Game Rules


- storywritter is a typing-based game made for 5 players.
- players take alternating turns, typing only one word at a time.
- each player has 8 seconds to type the word.
- if the player decides to end the sentence, he should put a punctuation mark, but doing so he can't write anything else.
- the goal of the game is to write a story and have fun.
- the game ends when the majority of players decide the story should be finished(by clicking the end-game vote button).
